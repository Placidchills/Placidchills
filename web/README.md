# Placidchills Web App

Next.js application for the Placidchills producer platform.

**Full project documentation lives in the repository root:**

- [`../docs/`](../docs/) — architecture, sprints, ADRs, coding standards
- [`../README.md`](../README.md) — quick start and deployment
- [`../CONTRIBUTING.md`](../CONTRIBUTING.md) — branch strategy and PR workflow

## Commands

```bash
npm install
npx prisma migrate dev
npm run dev      # http://localhost:3000
npm run build
npm run lint
```

## Environment

Copy `.env.example` to `.env` and configure before running locally.
